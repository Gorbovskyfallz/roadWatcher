1. Разность массивов: даны два массива, нужно вычесть один из другого таким
образом, чтобы в полученном массиве не было цифр, которые бы присутствовали во
втором массиве

package kata

func ArrayDiff(a, b []int) (r []int) {
  m:=map[int]bool{}
  for _,v:=range b{
    m[v]=true
  }
  for _,v:=range a{
    if !m[v] {r=append(r,v)}
  }
  return // a-b
}
Пояснение: для определения тех чисел, которые должны присутствовать в конечном
массиве сначала создадим мапу, которая будет в себе сожержать два числа из
второго массива и булевые true

вторым циклом мы прогоняем первый массив и смотрим, есть ли в мапе подобное
значение, если его там нет, то добавляем в конечный массив число.
